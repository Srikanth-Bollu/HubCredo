## Step - 1 : Clone the project
### git clone <your-repo-link>

## Step - 2 : Go to backend folder & install all dependencies
### cd backend
### npm i

## Step - 3 : Add a .env file in backend root folder
### Add all values exactly as shown in the .env.example file.

## Step - 4 : Run the backend
### npm start

## Step - 5 : Go to frontend folder & install all dependencies
### cd ../frontend
### npm i

## Step - 6 : Run the frontend
### npm run dev
